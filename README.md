# FVCI Community Platform

A prototype community platform built for the Idaho Veterans Coalition as part of a Software Engineering & Senior Design capstone project.

The project focuses on improving digital engagement and communication between members through:

* New member signup forms
* Mentor directory and contact system
* Community-focused website structure
* Improved accessibility and user experience

## Tech Stack

* HTML
* CSS
* JavaScript

## Purpose

This project was designed to help strengthen the organization’s online presence and create a more connected experience for veterans and community members.

## Features

* Responsive website design
* Member registration interface
* Mentor information and contact access
* Community engagement focused layout
